# 📚 Elite BookStore System

A fully functional **PHP & MySQL Bookstore Web Application** with admin panel, cart system, order management, and file upload (images & videos).

---

## 🚀 Features

### 👨‍💼 Admin Panel
- Add new books
- Edit existing books
- Delete books
- Upload book cover images
- Upload video previews
- Stock management

### 🛒 Customer Features
- Browse books
- Search & filter by category
- Add to cart
- Update cart quantity
- Checkout system
- Order tracking

### 📦 System Features
- Session-based cart system
- MySQL database integration
- File upload system (images/videos)
- Stock validation
- Order & order items tracking
